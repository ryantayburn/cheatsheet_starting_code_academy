CSS Display and Positioning Cheatsheet

CSS z-index Property

  • Description: Specifies how far back or forward an element appears. Useful for creating layered effects, such as modals or pop-ups
  • Example:
.element1 {
    position: absolute;
    z-index: 1;
}

.element2 {
    position: absolute;
    z-index: -1;
  }

Fixed CSS Positioning

  • Description: Pins elements to a specific spot. Commonly used for navigation bars.
  • Example:
#navbar {
  position: fixed;
  top: 0;
  width: 100%;
}

CSS Display Property

  • Description: Determines how an element is rendered.
  • Examples:
.container1 {
     display: block; 
} /* Full width */

.container2 {
    display: inline; 
} /* Inline with content */

.container3 {
    display: inline-block; 
} /* Inline with dimensions */

CSS Position: Absolute

  • Description: Positions an element relative to its nearest positioned ancestor.
  • Example:
.element {
  position: absolute;
  top: 10px;
  left: 20px;
}

CSS Position: Relative

  • Description: Positions an element relative to its normal position.
  • Example:
.element {
  position: relative;
  top: -10px; /* Moves 10px up */
}

CSS Position: Static

  • Description: Default positioning according to the document's flow.
  • Example:
.element {
  position: static; 
} /* Default */

CSS Float Property

  • Description: Floats elements left or right.
  • Examples:
.left {
    float: left;
    width: 50%;
}
.right {
    float: right;
    width: 50%;
}

CSS Clear Property

  • Description: Controls the flow of floated elements.
  • Examples:
.element {
    clear: left;
} /* Clears left floats */

.element {
    clear: right;
} /* Clears right floats */

.element {
    clear: both;
} /* Clears both sides */

Flexbox and Grid Layout

  • Description: Flexbox for one-dimensional layouts; Grid for two-dimensional.
  • Flexbox Example:
.container {
    display: flex;
    justify-content:
    space-between;
}
  • Grid Example:
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Use of `!important` with Positioning

  • Description: Overrides conflicting declarations.
  • Example:
.element {
    position: absolute !important;
}
